## 📂 Featured Projects

### 1️⃣ Clearance AI *(In Progress)*
> *AI-native SaaS for FDA 510(k) submission readiness*

- **Problem:** FDA 510(k) submissions are time-intensive, costly, and heavily reliant on navigating predicate device strategy — a process that remains largely manual today.
- **Solution:** Clearance AI helps medical device companies and regulatory consultants prepare faster with **trust-first pathway analysis** and **intelligent predicate matching** against the FDA 510(k) database.
- **Core Features:** Device classification engine · Predicate intelligence search · Submission gap analysis · Regulatory pathway recommender.
- **Tech Stack:** Python, LangChain, LLMs, Spring Boot, React, PostgreSQL.
- **Status:** Core architecture & predicate intelligence engine in development.

### 2️⃣ Medical Coding Pilot *(Shipped)*
> *LLM-powered clinical code automation*

- **Impact:** Built a pilot system that leverages large language models to suggest accurate **ICD-10 and CPT codes** from clinical notes, reducing manual coding time and human error in medical billing workflows.
- **Key Design Decisions:** Prompt engineering for clinical specificity · Confidence scoring for auditor review · Structured output for EHR integration.
- **Tech Stack:** Python, LLMs, REST APIs, PostgreSQL.

### 3️⃣ Real-Time WebSocket Chat Application

- **Tech Stack:** Java 17+, Spring Boot, WebSocket, Maven.
